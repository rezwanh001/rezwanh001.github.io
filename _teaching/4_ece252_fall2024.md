    <h4><i class="fa-solid fa-link"></i> Course Description</h4>
    <a href="https://uwflow.com/course/ECE252" target="_blank">View on UWFlow</a><br><br>
    This course covers processes and threads (pthreads), system calls, concurrency (semaphore, mutex, monitors, and barrier synchronization), user-level memory management, performance and correctness of concurrent systems, deadlock detection and recovery, and file systems.<br>

    <h4><i class="fa-solid fa-bullseye"></i> Course Objectives</h4>
    <strong>Lecture Component — Students will be able to:</strong>
    <ul>
      <li>Invoke system calls to access resources like files, IPC mechanisms, and the network.</li>
      <li>Analyze, design, implement, and debug concurrent C programs using concurrency control constructs.</li>
      <li>Apply standard concurrency scenarios such as producer-consumer, readers-writers, and dining philosophers.</li>
    </ul>
    <strong>Lab Component — Students will be able to:</strong>
    <ul>
      <li>Interact with the Linux OS through the CLI and the C language API.</li>
      <li>Read documentation and interact with available libraries for multi-threading, file handling, and network operations.</li>
      <li>Use inter-process communication mechanisms such as shared memory.</li>
      <li>Identify, analyze, and solve concurrency and synchronization problems.</li>
    </ul>
